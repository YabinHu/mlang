Welcome to Mlang.  This is a compiler front-end for the m-language used by
Matlab/Octave like numerical computing softwares which is built on top of the
LLVM compiler infrastructure project.

If you're interested in more (including how to build Mlang) it is best to read
the relevant web sites.  Here are some pointers:

Information on Mlang:              http://mlang.llvm.org/
Building and using Mlang:          http://mlang.llvm.org/get_started.html
Information on the LLVM project:   http://llvm.org/

If you have questions or comments about Mlang, a great place to discuss them is
on the Mlang development mailing list:
  http://xxx-mlang-dev
